Web Application Programming and Hacking

Front-end Web Development Lab

Instructor: Dr. Phu Phong

Student: Isaac Jacobsohn

In this course, we study basic web application development with front-end (HTML5, JavaScript, CSS) and back-end (PHP/MySQL). Web application vulnerabilities and attacks are introduced and explored with hands-on exercises. Secure programming principles and practices will be introduced to avoid potential web application vulnerabilities and attacks.

Lab 1

In this lab, we explored the HTTP protocol using Wireshark. We also explored different packaged with C CGI Web applications, echo Web applications in PHP, and curl POST requests.

Lab 2

In this lab we created web pages as well as performed front end programming for said web pages using tools such as JavaScript, Ajax, jQuery, and Web API integrations.
